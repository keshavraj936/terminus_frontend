# Terminus Frontend

This is the React + TypeScript frontend for the Terminus platform. 

## 🚀 Quick Start
To run the full stack, please refer to the main **[README.md](../README.md)** located in the root of the `college-projects` folder! It contains detailed instructions for starting the MySQL database, the PHP backend, and this React frontend together.

If the backend and database are already running, you can start the frontend normally:
```bash
npm run dev
```
And view it at `http://localhost:5173`.
